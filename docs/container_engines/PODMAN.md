# Docker

Section talks about setting up and installing the podman engine on a computer with a debian-based OS.

Instructions as well as other methods of installation can be found [here](https://podman.io/docs/installation).

Last Updated: `16/04/2024`.

## Installation

Instructions below require `root` privileges.

Run `sudo su` or `su -` beforehand.

```sh
#!/bin/bash

apt-get update
apt-get install podman
```
